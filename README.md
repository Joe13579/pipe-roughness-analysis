Repo allows for data to be entered into arrays, and automatically will generate graphs that are curve fitted.

Saves the data entered to csv and saves graphs and csv to desired path in local device.


Requirements:

Python environment such as Spyder

SciPy

Pandas

NumPy

matplotlib

How to produce graphs:


Enter experimental data and error to arrays, then run and graphs produced.

CI, R^2 and value for roughness are printed in IDE.
